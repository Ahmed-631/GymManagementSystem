# Gym Management System

A full-featured **ASP.NET MVC Gym Management System** designed to manage gym operations efficiently. This application allows you to handle **members, trainers, workout plans, and training sessions** with a clean, maintainable, and scalable architecture.

---

## Features

- **Members Management:** Add, update, delete, and view gym members.  
- **Trainers Management:** CRUD operations for trainers and their profiles.  
- **Workout Plans:** Create and assign personalized workout plans to members.  
- **Training Sessions:** Schedule and track training sessions for members.  
- **Authentication & Authorization:** Secure access for admins, trainers, and members.  

---

## Architecture & Design

- **3-Layer Architecture:** Presentation Layer (MVC), Business Logic Layer (BLL), Data Access Layer (DAL)  
- **Generic Repository Pattern & Unit of Work:** Clean data access and management.  
- **Dependency Injection (DI):** Promotes loose coupling and testability.  
- **Separation of Concerns:** Each layer handles a specific responsibility for maintainability and scalability.  

---

## Technologies Used

- **Backend:** C#, ASP.NET MVC, Entity Framework  
- **Frontend:** HTML5, CSS3, Bootstrap, Razor Views  
- **Database:** SQL Server  
- **Design Patterns & Principles:** Generic Repository, Unit of Work, Dependency Injection, 3-Layer Architecture
